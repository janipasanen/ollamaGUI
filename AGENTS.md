# Development Instructions

## Workflow
1. Develop milestone by milestone.
2. Implement issues one by one.
3. Every feature must have a corresponding test to verify functionality and prevent regression.
4. Run lint and type-check before finalizing any task.

## Quality Assurance
- All new features must be verified with automated tests.
- UI components should be tested for responsiveness and cross-platform compatibility.
- API interactions with Ollama must include error handling and timeout tests.
